# DecodeLabs Internship - Project 2
## Exploratory Data Analysis

For this project, I analyzed the cleaned dataset to find patterns and trends.

What I looked at:
- Basic statistics (average, median, min, max)
- Product popularity
- Payment methods
- Order status
- Referral sources
- Outliers

What I found:
- Only 19% of orders are delivered. 81% are cancelled, returned, pending, or shipped.
- Average order value is $1,053.97
- Printers are the most popular product (181 orders)
- Online payments are most common (258 orders)
- Instagram drives the most traffic (259 orders)
- 8 extreme orders over $3,300
- 34 orders over $3,000

Tools:
- Python with Pandas
- VS Code
- Git

Files:
- analysis.py - My EDA script
- observations.txt - Detailed notes

How to run:
pip install pandas numpy
python analysis.py
